## Start the shopping process

![Click any product to open the product detail page](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-05.png)

*Click any product to open the product detail page*

![On the product detail page, the user selects size, color, and quantity, then chooses add to cart or buy now](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-06.png)

*On the product detail page, the user selects size, color, and quantity, then chooses add to cart or buy now*

![If the user adds the product to the cart, clicking the cart icon on the navbar opens the cart page](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-07.png)

*If the user adds the product to the cart, clicking the cart icon on the navbar opens the cart page*

![Choose checkout now to buy the product, then the user is redirected to the order confirmation page](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-08.png)

*Choose checkout now to buy the product, then the user is redirected to the order confirmation page*

![On the order confirmation page, the user enters personal information, selects a payment method, then places the order and pays](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-09.png)

*On the order confirmation page, the user enters personal information, selects a payment method, then places the order and pays*

![The user is redirected to the order detail page. If the payment method is MoMo or bank transfer, after the payment worker returns the payment result, the order status is updated to paid](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-10.png)

*The user is redirected to the order detail page. If the payment method is MoMo or bank transfer, after the payment worker returns the payment result, the order status is updated to paid*

![The user receives the following notification email](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-11.png)

*The user receives the following notification email*

![The user tracks the status of all orders on the My Orders page](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-12.png)

*The user tracks the status of all orders on the My Orders page*

![The user tracks the status of all orders on the My Orders page](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-13.png)

*The user tracks the status of all orders on the My Orders page*

![The user receives an email notification when the order status changes to shipping](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-14.png)

*The user receives an email notification when the order status changes to shipping*

![When the admin changes the order status to shipping, the order detail page allows the user to confirm “I have received the order” to complete the purchase process.](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-15.png)

*When the admin changes the order status to shipping, the order detail page allows the user to confirm “I have received the order” to complete the purchase process.*

![The user receives a thank-you email after completing the purchase process](/fcj-workshop-hungdung/images/5-Workshop/hashop-demo/demo-16.png)
