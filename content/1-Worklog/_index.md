In this worklog section, I present the learning and practice process of the First Cloud AI Journey program. The worklog was completed in 12 weeks, from 17/04/2026 to 09/07/2026, excluding Saturdays and Sundays.

The worklog content focuses on learning AWS foundational services, practicing labs by topic, and recording achievements each week. Key topics include: AWS Free Tier, IAM, VPC, EC2, RDS, Database Migration, Auto Scaling, CloudWatch, AWS Backup, IAM Identity Center, AWS Budgets, and AWS architecture summary.

**Week 1:** [Getting familiar with AWS, AWS Free Tier, AWS Console, and AWS CLI](1.1-Week1/)

**Week 2:** [Learning AWS IAM and access management](1.2-Week2/)

**Week 3:** [Learning Amazon VPC and AWS Networking components](1.3-Week3/)

**Week 4:** [Learning Amazon EC2 and basic server deployment](1.4-Week4/)

**Week 5:** [Learning advanced EC2, Backup, Storage Gateway, and S3 Static Website](1.5-Week5/)

**Week 6:** [Learning Amazon RDS and deploying Database on AWS](1.6-Week6/)

**Week 7:** [Practicing Database Schema Conversion & Migration with AWS SCT and AWS DMS](1.7-Week7/)

**Week 8:** [Learning Elastic Load Balancing and Auto Scaling Group](1.8-Week8/)

**Week 9:** [Learning Amazon CloudWatch, Monitoring, and Logging](1.9-Week9/)

**Week 10:** [Learning IAM Identity Center and centralized access management](1.10-Week10/)

**Week 11:** [Learning AWS Budgets and cost control on AWS](1.11-Week11/)

**Week 12:** [Summarizing knowledge, completing AWS architecture, and cleaning up resources](1.12-Week12/)
